# Workbench Builder

Build new ICM workbenches for any domain through a multi-stage guided process.

## Task Routing

| Task Type | Go To | Description |
|-----------|-------|-------------|
| Discover a workflow | `stages/01-discovery/CONTEXT.md` | Conversational exploration of the domain workflow |
| Map stage contracts | `stages/02-mapping/CONTEXT.md` | Formalize inputs, process, outputs for each stage |
| Scaffold the workbench | `stages/03-scaffolding/CONTEXT.md` | Generate the complete folder structure and files |
| Design onboarding | `stages/04-questionnaire-design/CONTEXT.md` | Build the setup questionnaire for the new workbench |
| Validate the workbench | `stages/05-validation/CONTEXT.md` | Verify conventions and cross-references |
